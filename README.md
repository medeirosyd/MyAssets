# MyAssets

MyAssets is a CLI-based inventory management tool written in Python.

## What It Is

MyAssets is a terminal-only application designed to manage assets through a simple, letter-based menu system.  
It allows users to store, view, modify, and export inventory data using CSV files.

## How to Run

Requirements:
- Python 3.x

Run the program from the terminal:

myassets.py

## What It Does

* Add inventory items
* Edit existing items
* Delete items with confirmation
* List all inventory items
* Print an inventory report to the console
* Store inventory data in a local CSV file
* Export inventory data to CSV
* Safely handle invalid user input
